# Project Screens / Screenshots Folder

Drop your high-quality project screenshots in PNG format directly in this directory:
- `syllabix.png` for Syllabix (AI-assisted content platform)
- `airis.png` for Airis (AI camera application)
- `aura.png` for Aura Security (Decentralized access system)
- `wema.png` for Wema (Decentralized grid energy marketplace)

Make sure they are named exactly as shown above. The app will automatically render these screenshots in both standard view and the expand-to-lightbox inspect view!
